Nimgame 2 Planetoids
====================

A demo game for the [Nimgame 2](https://github.com/Vladar4/nimgame2) engine.

Controls:
---------

| Action                    | Control             |
| ------------------------- | ------------------- |
| Aim                       | Mouse               |
| Fly                       | Right mouse button  |
| Shot                      | Left mouse button   |
| Pause                     | P                   |
| Toggle collider overlays  | F10                 |
| Toggle info panel         | F11                 |
